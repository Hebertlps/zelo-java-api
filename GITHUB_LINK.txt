Por questões de segurança, as credenciais do banco de dados Oracle não foram "hardcoded" no código-fonte.
Para executar o projeto localmente, configure as seguintes variáveis de ambiente:

## Como Clonar e Executar
git clone https://github.com/Hebertlps/zelo-java-api.git
cd zelo-java-api/zelo-api
./gradlew bootRun

A API estará disponível em http://localhost:8080
Swagger em http://localhost:8080/swagger-ui.html
